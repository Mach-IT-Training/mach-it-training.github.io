Lesson 1: The Four Functions of Computing

Core Principle: At a fundamental level, computers perform four essential operations: INPUT, PROCESSING, OUTPUT, and STORAGE.

✅ Concept Check

Question: Which component is responsible for PROCESSING in a computer system?

A) Hard Disk Drive
B) Central Processing Unit (CPU)
C) Random Access Memory (RAM)
D) Graphics Card
Explanation: The CPU (Central Processing Unit) is the "brain" of the computer responsible for executing instructions and performing mathematical and logical operations on data. RAM provides temporary storage, the hard drive provides long-term storage, and the graphics card handles video processing.

Lesson 2: Understanding Data Units of Measure

Data Measurement Hierarchy: Starting from the smallest unit (bit), data scales through bytes, kilobytes, megabytes, gigabytes, terabytes, to petabytes, with each step representing approximately 1,000 of the previous unit.

📋 Scenario

A client is confused about storage capacity when purchasing a new computer. They need to understand data units to make an informed decision about whether a 512GB SSD is sufficient for their needs.

Lesson 3: Processing Speed - Megahertz vs. Gigahertz

Hertz Explained: Hertz (Hz) measures the number of electronic cycles per second. Megahertz (MHz) = 1 million cycles/second. Gigahertz (GHz) = 1 billion cycles/second. Higher frequency means faster processing capability.

📋 Scenario

A customer is comparing two laptops: one with a 2.4 GHz processor and another with a 3600 MHz processor. They need help understanding which is faster and why processor speed matters.

Processor Speed Comparison

Megahertz (MHz)

1 MHz = 1,000,000 cycles/second

Example: 2400 MHz = 2,400,000,000 cycles/sec = 2.4 billion cycles/sec = 2.4 GHz

Gigahertz (GHz)

1 GHz = 1,000,000,000 cycles/second

Example: 3.6 GHz = 3,600,000,000 cycles/sec = 3600 MHz
Real-World Impact: A 3.6 GHz processor can execute 3.6 billion cycles per second, making it faster than a 2.4 GHz processor. However, clock speed isn't the only factor in overall performance—core count, cache size, and architecture also matter.

✅ Concept Check

Question: What does a processor speed of 3.2 GHz indicate?

A) The processor can store 3.2 gigabytes of data
B) The processor executes 3.2 billion cycles per second
C) The processor has 3.2 processing cores
D) The processor uses 3.2 watts of power
Explanation: GHz (gigahertz) measures clock speed—the number of cycles per second. A 3.2 GHz processor completes 3.2 billion cycles every second. This is different from core count, storage capacity, or power consumption.

Lesson 4: Motherboards, CPUs, and RAM

The Three Pillars: The motherboard provides electrical pathways connecting all components. The CPU performs mathematical and logical operations. RAM provides temporary, high-speed storage for active processes.

📋 Scenario

You're upgrading a workstation and need to explain to a junior technician why all three components—motherboard, CPU, and RAM—must be compatible and work together seamlessly.

Component Roles and Interactions

🔌 Motherboard

Function: System backbone

  • Connects all components
  • Provides power distribution
  • Houses chipset for communication
  • Contains expansion slots

🧠 CPU (Central Processing Unit)

Function: Processing engine

  • Executes program instructions
  • Performs calculations
  • Makes logical decisions
  • Controls system operations

⚡ RAM (Random Access Memory)

Function: Temporary workspace

  • Stores active program data
  • Provides fast data access
  • Volatile (cleared when powered off)
  • Works directly with CPU

✅ Concept Check

Question: What happens to data stored in RAM when you shut down your computer?

A) It is permanently saved to the hard drive
B) It is transferred to the CPU cache
C) It is completely erased because RAM is volatile
D) It remains in RAM until overwritten
Explanation: RAM is volatile memory, meaning it requires continuous power to maintain data. When the computer shuts down, all data in RAM is lost. This is why you must save your work to non-volatile storage (like a hard drive or SSD) before powering off.

Lesson 5: Understanding the BIOS

BIOS Role: The Basic Input/Output System (BIOS) is firmware that initializes hardware during boot-up and provides runtime services. It runs POST (Power-On Self-Test) and stores critical system configuration.

📋 Scenario

A computer won't boot, and you need to access the BIOS to troubleshoot. The user asks what the BIOS does and why it's important for system startup.

BIOS Boot Sequence

1. Power On

User presses power button

→

2. BIOS Loads

Firmware initializes from ROM

→

3. POST Runs

Tests hardware components

→

4. Boot Device

Loads OS from configured drive

BIOS Configuration Settings

Setting Category What It Controls
System Date & Time Sets the internal clock for the computer
Boot Order Determines which device to load the OS from (HDD, SSD, USB, DVD)
Memory Configuration Detects and configures installed RAM
CPU Settings Manages processor frequency and voltage
Security BIOS passwords, Secure Boot settings
Hardware Monitoring Temperature sensors, fan speeds, voltages

✅ Concept Check

Question: What is the primary function of POST (Power-On Self-Test) during the boot process?

A) To load the operating system into RAM
B) To test and verify that hardware components are functioning properly
C) To install device drivers for peripherals
D) To configure network settings
Explanation: POST (Power-On Self-Test) is a diagnostic routine run by the BIOS that checks the CPU, RAM, keyboard, disk drives, and other hardware to ensure they're working correctly before attempting to load the operating system. If POST detects a problem, it typically produces beep codes to indicate the issue.

✅ Concept Check

Question: Where would you configure which device the computer boots from (hard drive vs. USB drive)?

A) Operating System Control Panel
B) Device Manager
C) BIOS settings
D) Windows Registry
Explanation: The boot order (also called boot sequence or boot priority) is configured in the BIOS settings. This determines which storage device the system checks first when loading the operating system. You access BIOS during startup (typically by pressing Del, F2, F10, or Esc depending on the manufacturer).
